# site-shell-map

1. Sticky Overlay Container
   - 空間類型：Sticky
   - 結構特徵：外層為固定定位容器，內含全螢幕覆蓋選單、主導覽列與側邊拉出購物車。

2. Full-bleed Media Block
   - 空間類型：Full-bleed
   - 結構特徵：外層為滿版容器，內層含絕對排版的媒體輪播區塊與重疊的文字資訊層。

3. 2-Column Split Block
   - 空間類型：Split
   - 結構特徵：外層為基礎容器，內層為左右對切的雙欄佈局。

4. Horizontal Draggable Band
   - 空間類型：Card Band
   - 結構特徵：外層為橫向區塊，內含超出畫面的多個可橫向拖曳卡片容器。

5. Stacked Split List
   - 空間類型：Split
   - 結構特徵：頂部為左右分欄區塊，下方為多個由上而下垂直堆疊且個別皆為雙欄切分的內容項目。

6. Multi-column Grid Block
   - 空間類型：Footer
   - 結構特徵：外層為底部容器，內部由多個不同比例組合的分欄網格構成。

7. Fixed Absolute Layers
   - 空間類型：Sticky
   - 結構特徵：獨立於主內容流之外的固定定位層，包含全螢幕的訂閱表單與互動彈窗。
